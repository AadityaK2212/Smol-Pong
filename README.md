
# Smol-Pong
It is a mini console that can run a 2 player pong game using a coin cell battery and a 8X8 LED grid. ioriginally saw someoemake a smal buiness card that can play video games, so I elaborated off of that and made it to only play pong I also created one for flappy bird. It runs on an ATtiny 1616 Microcontroller.
